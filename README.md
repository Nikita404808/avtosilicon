# CS20 Frontend Clone

Воспроизведение публичных страниц cs20.ru на стеке Vue 3 + Vite + TypeScript. Проект имитирует структуру, дизайн и поведение витрины: каталог, карточка товара, новости, статические страницы, модальные окна, мини-корзина и пр.

## Требования

- Node.js ≥ 18

## Установка и запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173` в браузере.

Создайте `.env.local` с параметрами интеграции банка (см. `.env.example`):

```
VITE_BANK_API_BASE=https://api.ozonbank.ru
```

### Локальная авторизация

- Пользователи и хэши паролей сохраняются в `localStorage` (`cs20.auth.users`).
- Токен сессии хранится в `cs20.auth.session`, восстановление пароля — через `cs20.auth.resetTokens`.
- Сброс пароля осуществляется по ссылке `http://localhost:5173/reset-password?token=<token>`.

## Команды

- `npm run dev` — локальная разработка.
- `npm run build` — сборка продакшн-бандла.
- `npm run preview` — предпросмотр собранного бандла.
- `npm run lint` — ESLint c Vue/TypeScript правилами.
- `npm run format` — форматирование Prettier.

При первом `npm install` автоматически ставятся husky-хуки. Пре-коммит запускает `lint-staged` (ESLint + Prettier на изменённых файлах).

## Архитектура

- `src/main.ts` — инициализация приложения, Pinia, маршрутизация.
- `src/router` — конфигурация страниц, обновление `<title>`.
- `src/stores` — Pinia-хранилища (`cart`, `auth`, `catalog`).
- `src/pages` — страницы (home, каталог, карточка товара, новости, about, contacts, инфостраницы).
- `src/components` — атомарные/составные блоки (хедер, футер, модалки, фильтры, карусели).
- `src/mocks` — JSON-заглушки для товаров, категорий, брендов и статей.
- `src/styles/tokens.scss` — токены дизайна (spacing, типографика, цвета).
- `public/placeholder` — моковые изображения/иконки (подставляются в карточки, карусели и пр.).

## Настройки

- Vite alias `@` указывает на `src`.
- SCSS автоматически подключает `tokens.scss`.
- Все стили компонентах опираются на CSS custom properties (никаких «магических» чисел).
